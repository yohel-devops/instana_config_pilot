# Presentation

This folder contains the project pitch material for Instana Config Pilot.

## Files

- `Instana-Config-Pilot-pitch.md` - Slide-by-slide pitch content and speaker flow.

## Suggested Use

Use the Markdown file as:

- a speaking script,
- source material for slides,
- or a backup presentation outline during judging.

## Recommended Slide Timing

For a 3-5 minute pitch:

1. Problem - 30 seconds
2. Solution - 45 seconds
3. IBM Bob usage - 45 seconds
4. Hackathon pivot - 30 seconds
5. Demo workflow - 60 seconds
6. Impact and next steps - 30 seconds

Keep the live demo focused on the frontend workflow:

- load template
- detect sensors
- configure tags and IBM MQ
- generate YAML
- generate `.env.example`
- show compare mode
